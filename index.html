How to customize

  1. Clone the repo
  2. Make sure you have Node.js, Grunt and Ruby installed.
  3. Install the SASS plugin for Ruby: gem install sass
  4. Install the SASS plugin for Grunt: npm install grunt-contrib-sass --save-dev
  5. From the command line cd into the root of the buttons directory.
  6. Run npm install or sudo npm install (depending on your system permissions).
  7. On the command line run grunt dev, this will open a browser with buttons.
  8. Locate scss in the root directory.
  9. You can modify the _options.scss where you can customize colors, typography, etc …
  10. Anytime you save your changes the buttons showcase page will live reload with your changes!
